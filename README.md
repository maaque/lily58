# Programmiererfreundliches QWERTZ-inspiriertes Lily58 Layout

[English version below](#english-version)

Dieses Layout versucht eine programmiererfreundliche Version des QWERTZ-Layouts für das Lily58 abzubilden.

## Besonderheiten

### Umlaute

Umlaute sind auf denselben Tasten wie die dazugehörigen Vokale und mit LOWER ansprechbar. Ü würde also `u + SHIFT + LOWER` geschrieben werden. Für ß gilt das gleiche.

### Vim-style Pfeiltasten

LOWER + hjkl triggert die Pfeiltasten, was eine Vim-style Cursorbewegung in allen Programmen ermöglicht.

### Klammern auf den Innentasten

Die Klammern ({[ und )]} sind mithilfe der Layershifts und den oberen Innentasten direkt erreichbar.

### Mediakeys und Bluetooth

Bluetooth findet sich in der linken oberen Reihe mithilfe des UPPER-Modifiers.
Mediakeys darunter.

### Numpad

Die Numpadtasten finden sich auf der rechten Seite mithilfe des UPPER-Modifiers.

### Lock

Eine Lock-Taste (Bildschirmsperre) befindet sich auf der linken unteren Innentaste mit dem LOWER-Modifier.

# English Version

# Programmer-Friendly QWERTZ-Inspired Lily58 Layout

This layout attempts to create a programmer-friendly version of the QWERTZ layout for the Lily58.

## Features

### Umlauts

Umlauts are located on the same keys as their corresponding vowels and can be accessed with LOWER. For example, Ü would be typed as `u + SHIFT + LOWER`. The same applies to ß.

### Vim-Style Arrow Keys

LOWER + hjkl triggers the arrow keys, enabling Vim-style cursor movement in all programs.

### Brackets on Inner Keys

The brackets ({[ and )]} are directly accessible using the layer shifts and the upper inner keys.

### Media Keys and Bluetooth

Bluetooth can be found in the upper left row using the UPPER modifier.
Media keys are below that.

### Numpad

The numpad keys can be found on the right side using the UPPER modifier.

### Lock

A lock key (screen lock) is located on the lower left inner key with the LOWER modifier.
